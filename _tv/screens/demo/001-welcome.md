This card is a file: `_tv/screens/demo/001-welcome.md`.

The **renderer** (server.js) is watching this folder. The **agent** never watches anything — it just writes files here. Try it:

- Edit this file and save — the card updates instantly.
- Drag any card; the position is saved to `_layout.json`, not to these files.

So an agent rewriting content can never clobber your layout.
